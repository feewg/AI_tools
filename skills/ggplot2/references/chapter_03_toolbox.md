# Layers Overview

## Layer Purposes

Layers in ggplot2 serve three main purposes:

**1. Display the Data**
Plot raw data for pattern detection. Essential in early data exploration.

**2. Statistical Summary**
Display model predictions in context of data. Helps improve models and reveal data subtleties.

**3. Metadata**
Add context, annotations, and references. Can be background (under data, minimally perceptible) or foreground (highlighting important features).
